# Simple JavaScript Firebase Wrapper
## Allows for CRUD functions with Firebase Cloud Firestore


* Created with Firebase 6.1.1

__Firebase Cloud Firestore Data Organization Structure:__

```bash
Collection
    │
    └─Document
      │  │
      │  └Collection
      └Field  │ 
      │       └Document      
      └F...    │      │
      │        └Field └Collection   
      └F...    │
               └F...

```
